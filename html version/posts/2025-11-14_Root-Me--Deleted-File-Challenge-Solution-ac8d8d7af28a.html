How I found the flag

The evidence was a .image file, so I opened it using FTK Imager.

Add Evidence Item → choose Image File → load usb.image.

FTK displayed the available partitions and file systems, including unallocated space.

Inside the root directory, there was only one file: anonyme.png, an image of a tree.

Since it was the only file, I figured the metadata might be the key.

I extracted the PNG and ran exiftool on it — and found the creator’s name.

Then I tested different flag formats until I found the exact one the challenge required.

  • Javier_Turcot
  • Turcot_Javier
  • Turcot Javier
  • Javier Turcot
  • javier turcot

I verified it using another metadata viewer, which confirmed the same information.

I also used jimpl.com to check the file’s metadata.
 Correct flag: javier_turcot
 It took a few attempts since the challenge was case-sensitive and very specific.
