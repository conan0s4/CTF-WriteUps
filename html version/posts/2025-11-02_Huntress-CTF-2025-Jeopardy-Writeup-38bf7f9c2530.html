Huntress CTF 2025 Jeopardy Writeup

1.)Darcy

clue: Darcy

While trying multiple solutions, I noticed “Darcs” might be a clue. A quick Google search revealed a site called Darcs, which turned out to be a tool.

https://darcs.net/

I opened the terminal, navigated to the Darcs file, and explored the tool. Using darcs --help, I checked the available commands, experimented with a few, and eventually ran darcs log. While reviewing the logs, I discovered the flag.

3.) Tabby’s Date

For this challenge, I used the tree command to see the directory structure and noticed a bunch of .bin files. I navigated to them and examined their contents using tools like strings, cat, and binwalk to search for the flag. Eventually, I opened the files in a hex editor, which made them much more readable, and that’s how I finally found the flag.

4.)RFC 9309

Did a quick Google search on RFC 9309 and found it refers to /robots.txt, which defines rules for web crawlers. I then checked the website’s /robots.txt file, noticed an interesting line, inspected it using the browser’s developer tools, and found the flag.
